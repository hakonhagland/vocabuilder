Foreign language input

Here are some examples of how to enable foreign language input.

Ubuntu

To enable korean hangul input, install the ibus-hangul package. Add Korean Hangul to your input sources from the Settings->Keyboard dialog. You should now be able to switch input sources with keyboard shortcut SUPER+Space.

MacOS

See Write in another language on Mac or this guide.
